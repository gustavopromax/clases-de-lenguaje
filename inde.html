1.¿Qué es un Protocolos Http?

Es el nombre de un protocolo el cual nos permite realizar una petición de datos y recursos, como pueden ser documentos HTML.

2.¿Qué es un Query String?

Es un término que generalmente se utiliza para hacer referencias a una interacción con una base de datos. Es la parte de la URL que contiene los datos que deben pasar a las aplicaciones web.

3.¿Qué es un Verbos HTTP?

Estos verbos indican qué acción queremos realizar sobre el servidor y son GET, POST, PUT, PATCH, DELETE, HEAD, CONNECT, OPTIONS y TRACE.

4.¿Qué es un DNS?

Es el sistema que hace posible que podamos navegar por Internet, puesto que apunta los dominios al servidor correspondiente.

5.¿Qué es un CDN?

Es una red de servidores interconectados que acelera la carga de las páginas web para las aplicaciones que tienen un uso intensivo de datos.
